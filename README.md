- **‚ö° High Performance**: Optimized C++ core with multi-threading support for maximum throughput.
- **üöÄ Hardware Acceleration**: Native CUDA and TensorRT integration for lightning-fast inference.
- **üõ† Comprehensive Toolset**:

  - **Face Enhancement**: Restore and sharpen facial details.
  - **Expression Restoration**: Recover original facial expressions.
  - **Super Resolution**: Upscale low-resolution images and videos.
- **üì¶ Batch Processing**: Efficiently handle mixed inputs of images and videos.
