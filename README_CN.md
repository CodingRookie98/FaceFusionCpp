## FaceFusionCpp

>   这个项目是开源项目[facefusion](https://github.com/facefusion/facefusion)的c++实现。

## 安装

请在[Release]()页面下载打包好的程序（暂时只提供Windows x64）。

另外的环境要求：

> - CUDA >= 12.2 (CUDA 12.5测试通过) (如果开启)
> - CUDNN >= 9.2 (测试通过)
> - Tensorrt >= 10.2（测试通过)(如果开启)
> - ffmpeg >= 7.0.2 (测试通过)(视频)

## 功能

1.  图片换脸
2.  图片脸部增强
3.  批量处理
4.  多线程加速
5.  TensorRT引擎加速（可开启引擎缓存）
